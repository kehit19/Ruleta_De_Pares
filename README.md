Ruleta de Pares
¡Bienvenido al proyecto llamado Ruleta de pares!, en lenguaje c++ , en si este proyecto sería un juego de ruleta.

---- objetivo ----
Uno de los objetivos puede ser el desempate entre amigos (por alguna apuesta), el que gana decide que se hace,etc .
Básicamente cosas relacionadas con ruleta pero con la diferencia de que solo hay 1 número para ganar 

---- Requisitos ----
- GCC / MinGW / Code::Blocks / Dev-C++ /visual studio code / JDoodle/G++
- Git / GitHub para versionado

---- Compilación ----
```bash o powershell
g++ RuletaDePares.cpp -o RuletaDePares.exe
